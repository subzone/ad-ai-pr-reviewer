How it works

Up and running in 3 steps

No extra infrastructure. No webhooks. Just YAML.

1

Store your secrets

Add your git host token and Anthropic API key (or cloud provider credentials) to an Azure DevOps variable group. Mark both as secret.

2

Add the task to your pipeline

Drop AiPrReviewer@1 into any pipeline YAML. Set action: reviewPR, point it at your repo, and enable enableAiReview: true.

3

Open a pull request

The pipeline triggers, fetches the diff, sends it to Claude, and posts a structured review comment on your PR — automatically.

trigger: none

pr:
  branches:
    include: [main]

variables:
- group: ai-reviewer-secrets   # GITHUB_PAT + ANTHROPIC_API_KEY

pool:
  vmImage: ubuntu-latest

steps:
- task: AiPrReviewer@1
  inputs:
    action: reviewPR
    provider: github
    accessToken: $(GITHUB_PAT)
    repository: myorg/myrepo        # ← change this
    prNumber: $(System.PullRequest.PullRequestNumber)
    enableAiReview: true
    aiApiKey: $(ANTHROPIC_API_KEY)
    aiModel: claude-sonnet-4-6
